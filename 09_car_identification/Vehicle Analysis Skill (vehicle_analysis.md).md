  
  
## Vehicle Analysis Skill (vehicle_analysis.md)  
  
**1. Overview**  
このスキルは、固定カメラで記録されたMP4動画から「車両の特定と詳細分析」を自動化します。動画から車両を検出し、その特徴（車種、色、メーカー）およびナンバープレート情報を構造化データとして抽出します。  
**2. Objectives**  
* **動画解析**: 指定フォルダ内のMP4ファイルから定期的にフレームを抽出。  
* **高精度車両検知**: YOLO11（最新版）を使用して、混雑した背景から車両のみを分離。  
* **詳細属性推論**: Qwen2.5-VLを用い、車両の外観特徴およびナンバープレートの読み取りを試行。  
**3. System Architecture**  

| Stage | Tool | Description |
| --------- | ------------ | ------------------------------------------------- |
| Ingestion | OpenCV (cv2) | 指定フォルダのMP4を読み込み、数秒おきにフレームをサンプリング。 |
| Detection | YOLO11n | Class 2 (car) または Class 7 (truck) を検知し、車両部分をクロップ。 |
| Reasoning | Qwen2.5-VL | クロップ画像から、車種・色・ナンバープレート情報を抽出。 |
  
  
**4. 主な修正内容の解説**  
* **YOLO11へのアップグレード**: 最新のYOLOモデルを使用し、検知精度を向上させています。  
* **ローカルファイル対応**: yt-dlpを削除し、cv2.VideoCaptureによってフォルダ内の.mp4ファイルを直接処理するように変更しました。  
* **プライバシーへの配慮**: 以前の交通解析ロジックに基づき、特定の車両を「加害者/被害者」と決めつけず、「当該車両（Involved Vehicles）」として客観的に分析するプロンプトを推奨します。  
* **リソースの最適化**: 航空機と同様、YOLOで車両のみを切り出してからMLLMに渡すことで、VRAM消費と処理時間を抑えています。  
この構成により、固定カメラ映像から効率的に車両データベースを作成することが可能になります。
