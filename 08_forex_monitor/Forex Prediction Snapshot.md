**Forex Prediction Snapshot**  
  
## Overview  
  
This skill is designed for the Kaggle Notebook environment to provide an instantaneous market snapshot and short-term price forecast. Unlike continuous monitors, this script executes a **single-run** process: it fetches real-time data, calculates a statistical forecast, and renders an interactive dashboard before terminating.  
  
## Functional Requirements  
  
### 1. Data Acquisition  
  
* **Library:** `yfinance` (Standard Kaggle utility, no API key required).  
* **Symbol:** `USDJPY=X` (USD to JPY Spot Rate).  
* **Interval:** 1-minute bars.  
* **Method:** A single fetch command retrieves the last 120 minutes of trading data to provide context for the model.  
  
### 2. Prediction Engine  
  
* **Forecast Horizon:** Predicts the next 60 minutes (1 hour) from the moment of execution.  
* **Algorithm:** `statsmodels.tsa.arima.model.ARIMA`.  
* **Optimization:** Utilizes fixed hyperparameters `ARIMA(5,1,0)` for rapid, low-latency processing without the need for extensive training phases.  
* **Reliability:** Includes a fallback mechanism that returns the last known price if the statistical model fails to converge.  
  
### 3. Static Dashboard  
  
* **Library:** `Plotly` (Graph Objects) for a high-fidelity, interactive UI.  
* **Visual Components:**  
* **Actual Data:** A solid blue line showing the price action of the previous 2 hours.  
* **Forecast Data:** A red dashed line showing the projected path for the next hour.  
* **Timeline Marker:** A vertical dashed line labeled "PRESENT" to clearly distinguish between historical facts and statistical projections.  
  
  
* **Theming:** Dark mode optimized for professional data science environments.  
  
## Technical Stack  
  
* **Language:** Python 3  
* **Data Source:** Yahoo Finance (via `yfinance`)  
* **Time Series Analysis:** `statsmodels`, `numpy`  
* **Visualization:** `Plotly`  
  
## Execution Logic  
  
1. **Initialize:** Install and import necessary dependencies.  
2. **Fetch:** Download the most recent 120 minutes of USD/JPY data.  
3. **Analyze:** Fit the ARIMA model to the closing prices and generate a 60-step forecast.  
4. **Render:** Generate an interactive Plotly figure with a timestamped title and disclaimer.  
5. **Exit:** The script completes its task and releases memory immediately after displaying the chart.  
  
## Important Notes  
  
* **Market Hours:** If executed during weekends or market holidays when the Forex market is closed, the script may return an error or empty data.  
* **Financial Disclaimer:** This tool is for educational and informational purposes only. The predictions generated by the ARIMA model are based on past price movements and do not account for fundamental news, economic events, or market volatility. **It does not constitute financial advice.**
