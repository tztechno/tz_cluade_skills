# 部分2: 速度推定ロジック (Speed Calculation)

## 役割
矩形の幾何学情報（位置の変化とサイズの比）から、実世界の速度（km/h）を算出する。

## 算出アルゴリズム（比率法）
1.  **移動量の正規化**: 
    - `ΔPixel / Width_Pixel` の比率を求める。これにより、カメラに近い車（大きく映る）と遠い車（小さく映る）のピクセル移動量の差を補正する。
2.  **実寸法パラメータ**: 
    - 一般的な乗用車の車幅（例: 1.8m）を定数として定義。
3.  **時間軸の統合**: 
    - 動画のFPS（Frames Per Second）を元に、フレーム間の経過時間を算出。
4.  **変換式**:
    - $Velocity = \frac{\Delta Pixel \times RealWidth}{Width\_Pixel \times \Delta Time}$

## 精度向上のための検討事項
- **平滑化**: 移動平均フィルタ等を用い、検知の揺らぎによる速度の跳ね上がりを抑制。
- **パース補正**: 道路が斜めに走っている場合、矩形の「幅」ではなく「奥行き」やホモグラフィ変換を用いた方が精度が高まる可能性がある。
