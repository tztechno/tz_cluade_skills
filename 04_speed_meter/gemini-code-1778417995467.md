# 道路車両速度計測システム (Overall)

## 目的
固定防犯カメラのmp4映像から、AIを用いて車両を検知し、その走行速度（km/h）をリアルタイムまたはバッチ処理で推定する。

## 処理パイプライン
1.  **入力**: mp4動画ファイル（固定画角）
2.  **車両検知**: YOLOモデルによる車両のバウンディングボックス(BB)抽出
3.  **トラッキング**: 同一車両のフレーム間紐付け（速度算出に必須）
4.  **速度演算**: BBの移動距離とサイズの比率に基づいた速度推定
5.  **出力**: 速度情報が重畳されたアノテーション済み動画 / ログデータ

## 技術スタック
- Python 3.10+
- OpenCV (動画処理、描画)
- Ultralytics YOLO (物体検知)
- NumPy (数学演算)
