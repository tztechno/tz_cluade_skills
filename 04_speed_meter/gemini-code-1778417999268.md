# 部分1: 車両検知 (Detection & Extraction)

## 役割
各フレームから車両の座標（x1, y1, x2, y2）と信頼度、クラスIDを取得する。

## 実装詳細
- **使用モデル**: YOLO11 (または最新の軽量モデル)
- **対象クラス**: car, truck, bus, motorcycle
- **抽出データ**: `[frame_id, track_id, x_center, y_center, width, height]`
- **最適化**: 
    - 不要なエリア（空や建物）を無視するためのROI (Region of Interest) 設定。
    - 推論速度向上のためのハーフ精度 (FP16) 利用。

## 課題
- 停車車両と走行車両の区別。
- 車両同士の重なり（オクルージョン）への対策。
